# Blake Pierce — 独立开发者 / 数据爱好者

用数据和技术做点有意思的东西。个人产品门户。

## 产品矩阵

### 逻辑透镜 LogicLens
基于历史赛事数据与机器学习模型的数据分析研究项目。覆盖五大联赛，每日更新。

### 典萃 ClassiCore
AI 深度阅读伴侣。上传 PDF 自动拆解，生成知识胶囊，一键输出多平台创作脚本。

### 净言 AI Cleaner
AI 文本润色工具。一键优化表达，去 AI 味，让内容更自然流畅。

### 上岸雷达
公务员考试岗位智能匹配与信息查询。

### 私域粉丝投票
比赛分享和私域投票数据收集，了解粉丝热度与倾向。

## 技术栈

- **框架**: React 19 + Vite 8
- **路由**: React Router v7
- **样式**: Tailwind CSS v3 + CSS Custom Properties
- **部署**: 阿里云 ECS (广州)

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
