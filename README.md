# HumanWrite 产品门户

逻辑驱动 · 技术赋能 · 工业级稳固

HumanWrite 产品矩阵门户网站，包含两款核心产品：

### 球之见 Insight
让店主获客更有底气 — 针对彩店店主、体育分析师打造的带教式分析管理工具。

### 典萃 ClassiCore
让知识创作效率提升 10 倍 — AI 驱动的深度阅读伴侣，基于 RAG 技术精准拆解书籍核心脉络。

---

## 技术栈

- **框架**: React 19 + Vite 8
- **路由**: React Router v7
- **样式**: Tailwind CSS v3
- **部署**: 阿里云 ECS (广州)

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
